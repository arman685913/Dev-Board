Question 1: What are the different ways to select an element in the DOM?
getElementById → selects a single element
getElementsByClassName, getElementsByTagName, getElementsByName → select multiple elements (HTMLCollection/NodeList)
querySelector → selects the first matching element
querySelectorAll → selects all matching elements (NodeList)
Question 2: What is the difference between innerHTML, innerText, and textContent?
innerHTML → gets/sets content including HTML tags
innerText → gets/sets only visible text
textContent → gets/sets all text, ignoring HTML tags
Question 3: What is event delegation in the DOM?
Event delegation is a technique where a single listener is added to a parent element instead of adding listeners to multiple child elements. The parent handles events that bubble up from its children.
Question 4: What is event bubbling in the DOM?
Event bubbling is a process where an event starts from the deepest target element and propagates up to its parent elements.
Question 5: How do you create, add, and remove elements using JavaScript?
// 1. Create element
const div = document.createElement('div');
div.textContent = "Hello";

// 2. Add element to DOM
document.body.appendChild(div); // or append()

// 3. Remove element
div.remove(); // or parent.removeChild(div)
